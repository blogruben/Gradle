# Ordenar funciones

```
gradle esImpar
    > Task :esImpar
    doFirst
    El numero 4 es par
    > Task :tarea1
    tarea1
```


```
gradle verOrdenado
    > Task :verOrdenado
    Hola 1
    Hola 2
    Adios 1
    Adios 2
```


```
gradle tasks  
    ....
    Prueba tasks
    ------------
    esImpar - Indica si un numero es impar
    tarea1 - Tarea1
    Verification tasks
    ------------------
    check - Runs all checks.
    test - Runs the unit tests.
    ....
```
